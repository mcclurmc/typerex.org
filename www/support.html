TypeRex is an Open-Source software

TypeRex is open-source software developed by OCamlPro and INRIA.

  • You can ask your questions about TypeRex on Stack Overflow. Make sure to tag your question with at least tags TypeRex and OCaml.
  • Report bugs on Github.
  • Fork on Github to contribute.

Support

OCamlPro provides commercial support on TypeRex. OCamlPro can help you if you are looking for:

  • training on TypeRex
  • targeted development efforts (typically for supporting new platforms)
  • and long-term commercial support
